With NKN's Metdata Editor you can create a valid ISO 19115 metadata record for your dataset or dataset series or a Dublin Core metadata record for your non-data research product (tool, visualization product, model, script, etc.). To get started, select the type of metadata record you would like to create in the Record Options dropdown above.

If you are using NKN to store your data, select NKN as Data Manager from the Load Defaults dropdown.

NOTE: You must complete all requred fields and will need to Save your record before you can Publish it.

Please contact NKN if you have additional questions or if the data associated with your record are larger than 10GB.
