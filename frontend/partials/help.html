Tips for using NKN's Metadata Editor:

Use the Load or delete an existing record dropdown to see a list of your existing records, the date they were lasted modified and published, and to load a record into the webform for editing or to delete the record. Once you have completed all required fields in the form (highlighted in red), you will have the option to submit your record to the NKN Portal for publication, select Publish Record to NKN Portal option under Record Options dropdown. Once submitted, we will review and publish your record. If you make changes to a record you have already submited, you will need to resubmit. To view your metadata record as an xml document, use the View XML as dropdown and select your preferred format (ISO, Dublin Core, or ESRI-ISO.

Special Instructions for MILES users:

Use the Load Defaults dropdown and select MILES to pre-load some of the fields with MILES-related content. Please add to these fields as appropriate: for example, you will likely want to add place and thematic keywords related to your specific dataset. The bounding coordinates provided here are for the entire state of Idaho; modify to fit the location of your dataset as appropriate. If you are using NKN to store your data, also select NKN as Data Manager from the Load Defaults dropdown. Please provide the first author's name and university affiliation toward the end of the use restrictions text where it currently reads "(Name, University Affiliation)".
